# 上下班自动统计系统

全自动统计上下班时间，无需手动操作，通过局域网设备扫描实现。

## 功能特性
- ✅ 零手动操作，全自动检测离家/到家
- ✅ 静态IP + MAC地址双校验，准确率高
- ✅ 低权限运行，无需root
- ✅ Docker容器化部署，7×24h稳定运行
- ✅ 网页配置界面，修改配置即时生效
- ✅ 数据持久化存储在SQLite
- ✅ 支持自定义时间窗口、扫描间隔等参数

## 部署方法
### 方式1：Docker Compose 一键启动
```bash
# 进入项目目录
cd attendance-tracker

# 启动服务
docker-compose up -d
```

### 方式2：直接运行
```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python main.py
```

## 访问地址
服务启动后访问：http://你的服务器IP:8000

## 配置说明
1. 打开网页后进入「系统配置」页面
2. 填写你的手机MAC地址和静态IP地址
3. 根据需求调整时间窗口参数
4. 保存配置后立即生效

### 手机设置要求
- 连接家庭WiFi时设置静态IP
- 设置固定MAC地址（避免随机MAC导致识别失败）
- 开启WiFi休眠时始终保持连接

### 路由器设置要求
- 关闭AP隔离/客户端隔离功能，允许局域网设备互访

## 检测逻辑
### 上班检测（07:40~08:40，可配置）
- 每分钟扫描一次设备
- 连续2次扫描不到设备 → 判定为已离家上班，记录上班时间
- 记录后当天不再重复检测上班

### 下班检测（20:50~00:30，可配置）
- 每分钟扫描一次设备
- 扫描到设备在线 → 判定为已到家，记录下班时间
- 记录后当天不再重复检测下班

## 数据导出
数据保存在 `data/attendance.db` SQLite数据库中，可直接导出为Excel/CSV。
